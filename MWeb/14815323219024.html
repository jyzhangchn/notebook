寄存器->主存储器: 存放
磁盘-->主存储器: ②敲入回车\n复制目标文件\n中代码和数据
Note left of USB控制器: ①键盘输入字符串\n"./hello"
Note right of 主存储器: hello代码
主存储器-->寄存器: ③将字符串\n"hello,world"\n的字节复制
寄存器->>图形适配器: ④复制到显示设备
Note right of 图形适配器: 显示到屏幕中

void goto_cond(int a,int *p)
{
    if(p==0)
        goto done;
    if(a<=0)
        goto done;
    *p+=a;
    
done:
    return;
}

int fun_b(unsigned x)
{
    int val=0;
    int i;
    for(i=0;i<32;i++)
    {
        val=(val<<1)|(x&0x1);
        x>>=1;
    }
    return val;
}
